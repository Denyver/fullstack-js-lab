Twinblades

Estilo: Siaahra.

Localização: Siaahra, Thundaera.

Atributo: Destreza.

Manusear Twinblades

Pode manusear Twinblades sem desvantagem. As Twinblades são espadas gêmeas, seguradas com uma mão enquanto giram rapidamente.

Retalho Giratório

As Twinblades são lâminas gêmeas usadas girando pelo usuário para acumular RG (Retalho Giratório).

Para cada RG acumulado, o usuário tem +2 de dano. O RG é resetado se o usuário parar de atacar ou errar. É possível usar a ação de movimento para ativar novamente o Retalho Giratório. RG é utilizado nas habilidades para amplificar o dano.

(Nível 3) Redemoinho Gêmeos

O usuário avança com as lâminas girando até o inimigo, usando a ação de movimento para atacar o inimigo à sua frente, causando Ataque Básico + 6 de dano. No final da ação, acumula +4 de RG.

(Nível 7) Dança das Lâminas

O usuário realiza uma sequência de ataques enquanto as lâminas giram em suas mãos, causando (Mod.Des / 2) Ataque Básico no inimigo. Para cada ataque realizado, acumula +2 de RG.

(Nível 7) Defesa Circular

Enquanto as Twinblades estiverem girando, o usuário pode usar a ação de movimento para defender ataques de curta a longa distância. Usa-se (Mod.DES + Proficiência) para defender, e essa ação acumula +2 de RG.

(Nível 10) Giro Impetuoso

O usuário realiza um grande giro com as lâminas ao redor, cobrindo uma área de 6 metros. Todo o RG acumulado é multiplicado por 3 e o usuário causa Ataque Básico em todos os inimigos na área. Todos os acúmulos de RG são resetados após essa ação.
